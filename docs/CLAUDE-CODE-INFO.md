# Esclarecimento sobre o Uso do Claude Code e LLMs

## Propósito do Projeto

Este projeto tem como finalidade exclusiva e estrita ajudar a nossa equipe e os usuários da comunidade a configurarem **skills** e interações baseadas em LLMs (Modelos de Linguagem Grande) no terminal. Nós fornecemos configurações, exemplos e templates de estrutura voltados a aumentar a produtividade e organização em ambientes de linha de comando.
